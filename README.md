# Portfolio Website

A modern, responsive portfolio website built as a practice project to improve frontend development skills
using modern web technologies and development workflows.

## About

This project was created to practice building a personal portfolio from scratch while learning and applying
core frontend concepts. It focuses on creating a clean user interface, responsive layouts, and modern development practices.

Although this is a learning project, I aim to follow professional project structure and coding standards.

## Feature

- Responsive design for desktop, tablet, and mobile
- Modern and clean UI
- Smooth animations and transitions
- Interactive navigation
- Project showcase section
- Skills section
- Contact section
- Fast development with Vite
- Organized and maintainable codebase

## Learning Objectives

This project helped me practice:

- Semantic HTML
- Modern CSS layouts (Flexbox & Grid)
- Responsive web design
- JavaScript DOM manipulation
- Component organization
- Asset management
- Vite development workflow
- Clean project structure
- Git & GitHub workflow

## Technologies Used:

- HTML5
- CSS3
- JavaScript (ES6+)
- Vite
